#### For Events:
- Event Title (required)
- Date (required)
- Location (e.g., "Conference Room A", "Auditorium")
- Organizer (person/department organizing the event)

#### For Trainings:
- Training Name (required)
- Start Date (required)
- End Date (required)

- Agenda/Topics (what will be covered)

#### For Meetings:
- Meeting Title (required)
- Date (required)
- Time (required)
- Location (physical or virtual location)
- Meeting Link (video conference URL, if applicable)
- Organizer (who is organizing)
- Agenda/Topics (discussion points)

4. **Click Save** to create the announcement

## Troubleshooting

**Q: I don't see the "New" buttons**
- A: Only Quality role users can create announcements. Check your role in the system.

**Q: Meeting link didn't open**
- A: Make sure you've pasted the complete URL including https://

**Q: Can't find an announcement**
- A: Announcements are sorted by date (newest first). Scroll down to find older announcements.
